项目名称: CSDN下载助手

功能: 通过分析CSDN的各个http协议，实现登录，下载，评价资源等功能。

开发环境: qtcreater2.8 + qt4.8.6 + win8.1

	 ( 已在qt5.3测试，可编译通过。)

申明: 捧着开源精神，与大家分享源码，本项目仅用来学习与交流，请不要用于其他用途。谢谢。


作者: wu5151

博客地址: http://blog.csdn.net/wu5151

本项目博客: http://blog.csdn.net/wu5151/article/details/47143401
		( 界面部分有所修改)

时间:2015.8.1

关于几个配置文件: config.in存放的是用户的账号信息，未作任何加密处理。
		  url.txt存放的是爬虫地址。比如c++/c 移动开发  等下载区地址。可自行修改，添加删除。
				请从此处自行获取http://download.csdn.net/category

		  used.txt 存放的是用户当前下载资源进度。 


待改善问题: csdn的验证码问题。 建议，通过定时器来处理。
	    关于验证码的显示，有待改善。
	    其他问题，自己修改。

修复: 验证码频繁出现问题。验证码对齐问题。爬虫地址重复问题。
