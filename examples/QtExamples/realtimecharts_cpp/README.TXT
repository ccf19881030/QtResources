Real-Time Chart Extra Sample Code Pack
=======================================
This is an extra sample code pack for ChartDirector. If you are new to 
ChartDirector, it is recommended you download ChartDirector from 
http://www.advsofteng.com/download.html to try it. It includes complete 
documentation as well as plenty of sample code designed as tutorials. This
sample code pack assumes you already have some understanding of ChartDirector.

For information about this sample code pack, please refer to: 

http://www.advsofteng.com/tutorials/extra.html
